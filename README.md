# Kubernetes Monitoring Platform

## Overview
A cloud-native monitoring platform built on Kubernetes using:

- Kubernetes
- Prometheus
- Grafana
- Alertmanager
- NGINX Ingress
- Google Microservices Demo

## Features

- Microservices deployment
- Cluster monitoring
- Pod monitoring
- Node monitoring
- Dashboard visualization
- Alert management

## Architecture

User
↓
NGINX Ingress
↓
Microservices Application
↓
Prometheus
↓
Grafana
↓
Alertmanager

## Technologies

- Docker
- Kubernetes
- Helm
- Prometheus
- Grafana
- Alertmanager
- NGINX Ingress
